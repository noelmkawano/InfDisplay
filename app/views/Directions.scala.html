Directions

This web app controls each of the 576 LEDs on the Infinity Display with a virtual canvas. The user will animate a frame, input a password, and submit the frame to the display. Each accepted submittal is added/combined from the previous frames and pushed to the display. The current version allows for one preset color (white) and all frames are wiped after 60 seconds.

Step 1: The Canvas

Users can access the canvas page through the home page, navbar, or this url, www.infdisplay.com/canvas

Step 2: Animating Frames

Users can click (for computers) or tap (for mobile) the array of boxes to switch the LEDs on or off. Black boxes represent an off LED, white represents a lit LED.

Step 3: Submittal

Users can submit their frames by entering the correct password and clicking the submit button at the top of the virtual canvas. The purpose of the password is to protect the display from any third party user from spamming. Ideally venues would release the password to significant customers.
